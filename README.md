# Life Debt（命债）

一个 Fabric 模组：**向死亡借命，用债务喂养世界的敌意。**

签下命债，你便能在致命一击下不死——代价是每借一次命，债务就攀高一分。债台越高，世界越不容你：先是缠身的粒子，接着是夜里循迹而来的追债者，最终整个世界都在向你讨债。你选择的**契约**，决定你以什么姿态偿还这笔债。

> 这是 `1.2.0` 起的重构方向。旧的「可食用图腾 + 命债增益 + 生命上限偿还」机制（1.x）已被下方的「契约 + 债务等级」框架取代。

## 核心循环

```
右键不死图腾 → 签订契约（获得借命容量）
      │
      ▼
受到致命伤 → 还有容量？── 是 → 借命抵死（回满血，容量−1，债务+1）
      │                          │
      否                         ▼
      │                  债务累积 → 跨越阈值 → 债务等级提升
      ▼                          │
   正常死亡                       ▼
                        世界影响加剧（粒子 → 追债者 → 世界异常）
```

- **签约**：手持不死图腾右键，消耗一枚图腾，签订契约并获得借命容量。
- **借命抵死**：拥有容量时受到致命伤不会死亡，而是回满血、容量 −1、债务 +1。容量耗尽后，命债不再介入，玩家正常死亡。
- **债台高筑**：债务只增不减（偿还机制规划中），并按阈值推导出**债务等级**。
- **世界追债**：债务等级越高，世界对你的敌意越强。

## 契约（Contract）

契约不是 Buff，而是你的玩法方向——决定借命的代价与偿还的姿态。首批三种：

| 契约 | 定位 | 设计意图 |
|---|---|---|
| **血契** | 狂战 | 生命越低，伤害与攻速越高；死亡增加大量债务。 |
| **魂契** | 法师 | 死亡时支付经验 / 物品 / 耐久，换取更低的生命损失。 |
| **亡契** | 逃亡 | 死亡时瞬移并短暂无敌，但增加被追债的概率。 |

## 债务等级（Debt Level）

债务等级由当前债务值派生（不单独存储，避免失同步）。跨越阈值即解锁对应的世界影响：

| 等级 | 债务阈值 | 世界影响 |
|---|---|---|
| 正常 | `0` | 无。 |
| 借命者 | `≥ 5` | 缠身粒子效果。 |
| 负债者 | `≥ 10` | 夜晚生成「追债者」。 |
| 亡命者 | `≥ 20` | 世界出现异常（村民涨价等）。 |
| 死人未亡 | `≥ 50` | 触发终极世界事件。 |

## 支持的版本

单一源码，通过 [Stonecutter](https://stonecutter.kikugie.dev/) 为每个 Minecraft 版本各产出一个 jar。

- **当前开发主节点：`1.21.1`**——新核心层在此编写并验证。
- 其他现代节点（`1.21.4` / `1.21.8` 等）因 **1.21.2+ 的 API 断代**（如 `UseItemCallback` 返回类型变化）尚在移植，会用 Stonecutter 条件注释逐版补齐。

> 26.x 因改动较大，暂缓，后续再评估。

加载器：Fabric Loader `>=0.19.3`（全版本通用）。需要 Fabric API。
