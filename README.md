# Life Debt · 命债

**简体中文** ｜ [English](./README.en.md)

> 透支来日之寿，换取此刻不死；债若不能偿，则当场燃尽成灰。

一个 Fabric 模组。吞下不死图腾，「命债」加身：其间凡致命之伤皆可赖过，越是濒死越是难死；然而每一次向死神赊来的苟活，终将在增益散尽之时，以**永久生命上限**偿还。倘若债台高筑、无以为继，便在原地燃尽。

## 核心机制

- **图腾可食**：不死图腾化作随身口粮（饥饿值 6、饱和度 1.0），一口吞下，换得**五分钟「命债」增益**。
- **抵死不亡**：命债在身时受到致命伤害不会殒命，而是触发一次与图腾等效的复生——回满生命、涤净所有减益，并获 再生 II（45 秒）、伤害吸收 II（5 秒）、抗火（40 秒）、夜视（3 分钟），以及在原有等级上再进两阶的**力量**（1 分钟）。每赖过一死，死亡计数加一。
- **越死越难死（叠乘减伤）**：减伤比例随赖死次数累积而叠乘——
  `R = 1 − 1 / (1 + k · D / H_max)`
  其中 `D` 为本次增益期间累计的死亡次数，`H_max` 为当前生命上限，`k` 为可调系数（默认 20）。实际受到的伤害 = 原伤害 ×（1 − R），并附带等比例的**击退抗性**（上限 100%）。
- **以命偿债**：增益散尽之际，按死亡次数**永久削去等量的生命上限**。
- **燃尽成灰**：若削减后生命上限降至零或更低，玩家以自定义的「燃尽」伤害类型陨落（死亡信息：「%s 燃尽了」），并解锁进度**「燃尽此身」**。
- **重生免债**：玩家重生之后，生命上限的惩罚一笔勾销。

死亡计数与会话状态皆借由玩家 NBT 持久保存。

## 配置

配置文件位于 `config/lifedebt.yml`（首次运行自动生成，内含公式注释）。

| 键 | 默认值 | 说明 |
|---|---|---|
| `damageReductionK` | `20.0` | 减伤系数 `k`。可由目标减伤反解：`k = H_max · R / (D · (1 − R))`。例如欲在 `H_max = 20、D = 10` 时达到 `R = 90%`，则 `k = 18`。 |
